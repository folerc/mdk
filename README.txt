Описание проекта.

Функция input_numbers():Запрашивает у пользователя ввод чисел до тех пор, пока не будет введено слово "стоп".Сохраняет введенные числа в списке numbers.

Арифметические функции:
add(numbers): Суммирует все числа.
subtract(numbers): Вычитает все последующие числа из первого.
multiply(numbers): Перемножает все числа.
integer_divide(numbers): Выполняет целочисленное деление.
exponentiate(base, exponent): Возводит основание в степень.
modulus(numbers): Вычисляет остаток от деления.

Функции для сравнения и проверки:
compare_two_numbers(a, b): Сравнивает два числа и возвращает результат.
is_prime(n): Определяет, является ли число простым.
find_min_max(numbers): Находит наименьшее и наибольшее число из списка.

Функция main(): Запускает процесс ввода чисел и выполняет арифметические операции, сравнивает числа, проверяет на простоту, и ищет наименьшее и наибольшее число.
